#  Langflow Project

##  Overview
A beginner Python project to manage contributors and issues in a simple open-source style system.

## Python Version
 - Tested with Python 3.13.3

##  Features
- Add contributors (name, role, language, commits, country)
- Track issues (ID, title, type, priority, status)
- Basic analysis (open issues, top reporter)
- Generate files:
  - project_report.txt
  - issues.csv

##  How to Run
python main.py

##  Output
Creates a folder with:
- project_report.txt
- issues.csv

##  Concepts Used
- Tuple, List, Dictionary, Set
- File handling
